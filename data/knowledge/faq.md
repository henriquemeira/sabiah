# Base de Conhecimento - Software Exemplo

## FAQ - Perguntas Frequentes

### Como resetar minha senha?

Para resetar sua senha, siga os passos:
1. Acesse a tela de login
2. Clique em "Esqueci minha senha"
3. Digite seu e-mail cadastrado
4. Você receberá um link de reset no seu e-mail
5. Clique no link e defina sua nova senha

### Como emitir uma nota fiscal?

Para emitir nota fiscal:
1. Acesse o menu "Financeiro" > "Notas Fiscais"
2. Clique em "Nova NF-e"
3. Preencha os dados do cliente
4. Adicione os produtos/serviços
5. Clique em "Emitir"

### Qual a versão atual do software?

A versão atual é **4.2.0**. Verifique sempre se seu sistema está atualizado em Configurações > Sobre.

### Como fazer backup dos dados?

O backup pode ser feito de duas formas:
- **Automático**: Configure em Configurações > Backup > Automático
- **Manual**: Acesse Configurações > Backup > Exportar

### Onde encontro o relatório de vendas?

O relatório de vendas está em:
- Menu "Relatórios" > "Vendas"
- Pode filtrar por período, vendedor, cliente

## Problemas Conhecidos

### Erro ao importar XML

**Sintoma**: Erro "XML inválido" ao importar nota fiscal

**Solução**: 
1. Verifique se o XML está no formato correto (NFe 4.0 ou superior)
2. Certifique-se que o arquivo não está corrompido
3. Tente abrir o XML em um editor de texto para validar

### Sistema lento

**Soluções**:
1. Limpe o cache: Configurações > Manutenção > Limpar Cache
2. Feche abas desnecessárias
3. Verifique sua conexão com a internet

## Tutoriais

### Primeiro acesso

Bem-vindo ao software! Este guia vai ajudá-lo:

1. **Cadastre sua empresa**: Configurações > Empresa > Dados
2. **Configure impostos**: Configurações > Fiscal > Impostos
3. **Crie usuários**: Configurações > Usuários > Novo

### Integração com WhatsApp

Para integrar com WhatsApp Business:
1. Acesse Configurações > Integrações > WhatsApp
2. Escaneie o QR Code com seu celular
3. Autorize a conexão
